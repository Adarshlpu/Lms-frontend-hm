#LMS Frontend


###setup  instruction 


1.clone the project
``` github-clone  https://github.com/Adarshlpu/Lms-frontend-hm.git
```


2.move into the directory
```
cd lms-frontend

```
3. install dependency

```
npm i
```

4.run the server

```
npm run dev

```
